# LI BAR / 理BAR

华东理工大学奉贤校区学生自运营鸡尾酒活动的单页网站。页面使用原生 HTML、CSS、Vue 3 和 Tailwind CDN 实现，包含活动展示、酒单展示、店庆转盘以及隐藏的点单统计系统。

## 功能概览

### 公开展示页

- 首页 Hero 展示 LI BAR / ECUST BAR 的主视觉图片和入口按钮。
- 顶部导航支持跳转到 Event、About、Menu 等页面区域。
- Event 区域展示当期主题活动，例如 Fruit Party。
- About 区域展示活动介绍、地址、下一次活动时间和 Bartender 信息。
- Footer 展示微信二维码、手机号复制、品牌信息和版权信息。
- 页面内图片带滚动进入动画和缩放过渡效果。
- 桌面端启用自定义鼠标光标和 hover 反馈。
- 移动端适配视口、触摸滚动和安全区域。

### 酒单系统

- 展示水果鸡尾酒和经典/改编鸡尾酒两大类。
- 每杯酒展示英文名、中文名、价格、风味标签、配料、描述和 ABV 信息。
- 支持 `Sold Out` 状态，售罄饮品会弱化显示并划掉价格。
- 经典鸡尾酒按基酒分组，包括金酒、伏特加、威士忌、朗姆和混合基底。
- 配料行会自动从描述文本中解析，并与正文说明分开展示。

### 店庆转盘

- 可通过 `WHEEL_ENABLED` 控制是否显示转盘入口。
- 转盘奖项包括 3 元、5 元、8 元、10 元优惠。
- 抽奖概率按区间配置：3 元 70%，5 元 20%，8 元 8%，10 元 2%。
- 点击开始后转盘旋转并在结束后高亮中奖项。
- 转盘页可返回主页。

### 隐藏 POS 点单系统

- 点击页脚 `© 2026 Li Bar.` 进入隐藏 POS 界面。
- POS 使用深色全屏界面，进入后会隐藏普通页面并禁止页面滚动。
- 按水果鸡尾酒和经典鸡尾酒分组显示可售饮品按钮。
- 单击饮品按钮打开配方图片预览。
- 双击饮品按钮添加一杯订单。
- 3 秒内再次添加同一饮品会撤销上一杯，方便误触回退。
- 每个饮品按钮显示该饮品当晚累计杯数。
- 订单按 5 分钟时间段生成横向时间轴。
- 时间轴中每个订单卡展示英文名、中文名和完成状态。
- 双击订单卡可切换完成 / 未完成。
- 移动端支持双击触发完成状态切换。
- 时间轴会自动滚动到最新订单位置。

### POS 数据统计

- 顶部显示当晚服务日期和总杯数。
- 数据使用浏览器 `localStorage` 持久化保存。
- 服务日期以当天 14:00 为分界；凌晨到 14:00 前仍计入前一服务日。
- 支持清空今晚数据，清空前会弹窗确认。
- 支持导出今晚数据到剪贴板。
- 导出内容包括总杯数、每款酒杯数，以及点单最多的前三个 5 分钟时间段。

### 配方图片预览

- POS 中点击饮品会尝试打开配方图。
- 图片路径会按多个候选地址查找，例如 `images/id-{id}.png`、`images/recipes/{id}.png` 等。
- 如果所有候选图片都不存在，会显示图片缺失提示。
- 预览弹窗支持关闭。

### 数据与部署

- 依赖通过 CDN 加载：Vue 3、Tailwind CSS、Google/Loli 字体。
- 包含 Vercel Analytics 和 Speed Insights 脚本。
- 使用 `favicon.png` 作为网页图标和 Apple Touch Icon。
- 图片资源默认从 `images/` 目录读取。

## 技术栈

- HTML5
- CSS3
- Vue 3 Composition API
- Tailwind CSS CDN
- Browser LocalStorage
- Clipboard API
- IntersectionObserver

## 运行方式

这是一个静态页面项目，直接打开 `index.html` 即可运行；如果部署到 Vercel、GitHub Pages 或其他静态托管服务，也不需要额外构建步骤。

## 主要文件

- `index.html`：页面结构、样式和全部交互逻辑。
- `images/`：活动图片、二维码和配方图片资源。
- `favicon.png`：站点图标。
